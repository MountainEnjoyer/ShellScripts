## Functions

  ```
  function function_name
  {
    commands/statements
  }
```
```
  function_name()
  {
    commands/statements
  }
```

## Case statements
  - Shell supports case...esac statement
  ```
  case word in 
   pattern1)
     statement;;
   pattern2)
     statement;;
   pattern3)
     statement;;
   *)
     Default statement;;
  esac
```
